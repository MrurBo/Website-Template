# Template website

This template is a template that combines scss with Jekyll.

## Configuration
1. Edit `_config.yml` with your details.
2. Edit `CNAME` if you have your own domain.
3. Edit `src/index.md`, `src/js/main.js`, and `src/_sass/*` for all styles.
4. When your done, push files to github and wait.

## Runs:
* Test: `jekyll serve`
* Build: `jekyll build`
* Deploy: Automatically via github actions.

## If this isnt for you, you dont have to use this.
